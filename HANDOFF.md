## What this repo is

`iot-pdm-pipeline` is a self-study prototype of an end-to-end
industrial-IoT predictive-maintenance pipeline:

```
synthetic vibration sensor
   -> MQTT broker (EMQX)
   -> Kafka topic
   -> TimescaleDB hypertable
   -> IsolationForest anomaly detector
   -> Grafana dashboard
```

GitHub: https://github.com/waynehacking8/iot-pdm-pipeline
